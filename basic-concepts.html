# Valid names my_variable = 10 _private = 5 name123 = "Alice" # Invalid names 123name = "Bob" # Can't start with number my-variable = 10 # Can't use hyphens my variable = 10 # Can't use spaces

Conventions

Style guidelines for making variable names consistent and readable

# snake_case (Python standard) user_name = "Alice" total_count = 42 # camelCase (used in other languages) userName = "Alice" # PascalCase (used for classes) UserProfile = {}

Python prefers snake_case (words separated by underscores, all lowercase)

Reassigning Variables

Changing a variable's value after it's been created

count = 5 print(count) # Output: 5 count = 10 # Reassign to new value print(count) # Output: 10

Assigning multiple variables in a single line

# Assign multiple variables at once x, y, z = 1, 2, 3 # Swap values a, b = 5, 10 a, b = b, a # Now a=10, b=5

Variable Types

Understanding different data types variables can hold

age = 25 # Integer (int) height = 5.9 # Float (decimal number) name = "Alice" # String (text) is_student = True # Boolean (true/false) numbers = [1, 2, 3] # List (collection)

Dynamic Typing

Python allows variables to change types

# Same variable, different types value = 10 # Integer value = "Hello" # Now it's a string value = 3.14 # Now it's a float

Note: This is unique to Python. Languages like Java don't allow this.

Type Casting

Converting one data type to another

# Convert string to integer num_string = "25" num = int(num_string) # Now it's 25 (integer) # Convert float to integer decimal = 3.9 whole = int(decimal) # Now it's 3 (truncated) # Convert to string age = 30 age_text = str(age) # Now it's "30" (string)

Empty Variable

Using None to represent the absence of a value

# Declare a variable with no value empty_var = None # Check if variable is None if empty_var is None: print("Variable is empty") # None is different from 0 or empty string zero = 0 empty_string = "" # These are NOT the same as None

Errors

Code errors are mistakes in your program that prevent it from running correctly or at all. There are several types:

1. Syntax Errors

Mistakes in how you write code (grammar of programming)

# Syntax Error - missing closing parenthesis print("Hello" # Syntax Error - missing colon after if statement if x > 5 print("x is big")

Result: Program won't run at all

2. Runtime Errors

Errors that occur while the program is running

# Runtime Error - dividing by zero result = 10 / 0 # Runtime Error - using undefined variable print(undefined_variable)

Result: Program crashes during execution

3. Logical Errors

Code runs without crashing, but produces wrong results

# Logical Error - wrong calculation total = 5 + 3 * 2 # Might give unexpected result if order of operations is misunderstood

Result: Program runs but gives incorrect output

4. Type Errors

Errors that occur when you try invalid type conversions

# Type Error - can't convert non-numeric string to integer result = int("hello") # ERROR! # This works because the string contains a number result = int("25") # Works fine, result = 25
