Basic Functions

# Function definition def greet(): print("Hello, World!") # Function call greet()

Functions with Parameters

def greet(name): print(f"Hello, {name}!") greet("Alice") # Hello, Alice!

Functions with Return Values

def add(x, y): return x + y result = add(5, 3) # 8

Multiple Parameters

def calculate_area(length, width): return length * width area = calculate_area(5, 10) # 50

Default Arguments

def greet(name="World"): print(f"Hello, {name}!") greet() # Hello, World! greet("Alice") # Hello, Alice!

Type Hints

def add(x: int, y: int) -> int: return x + y def greet(name: str) -> None: print(f"Hello, {name}!")

Variable Scope

# Global scope global_var = "I'm global" def my_function(): # Local scope local_var = "I'm local" print(global_var) # Can access global print(local_var) # Can access local my_function() # print(local_var) # Error: local_var not defined
